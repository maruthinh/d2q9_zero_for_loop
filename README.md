D2Q9 LBM code to solve lid driven cavity problem. This code has zero for loops

To run the code 

```
python3.8 lid_driven_cavity.py
```

**Sample results**

*Simulation parameters*
- Reynolds number: 100
- Mach number: 0.1
- Number of grid points: 64 x 64
